## 5. Forum: rennrad-news.de/forum

**Wann posten:** Erst wenn du ~10-20 hilfreiche Antworten in anderen Threads gegeben hast.  
**Kategorie:** Technik / Pflege & Wartung  
**Titel:** Heißwachs selbst gemacht — Erfahrungen nach 2 Jahren

```
Hallo zusammen,

ich wollte mal meine Erfahrungen mit Heißwachs teilen, nachdem ich seit 2023 
intensiv damit experimentiere.

Kurz zum Hintergrund: Ich produziere selbst ein Heißwachs (Waxcelerate) auf 
Paraffinbasis mit PTFE, aber das hier ist kein Werbethread — ich möchte einfach 
über die Technik reden.

Was ich gelernt habe:
- Entfetten vor dem ersten Wachs ist nicht optional. Werkfett und Paraffin vertragen 
  sich nicht. Isopropanol 99% reicht für die meisten, Ultraschall ist natürlich besser.
- 85-90°C ist der Sweet Spot. Darunter wird das Wachs zu zäh, darüber degradiert 
  es schneller.
- 3-Ketten-Rotation ist der Game Changer. Ohne Rotation ist Heißwachs ein Aufwand. 
  Mit Rotation: alle 15min pro Session, 3 Ketten gleichzeitig gewachst.
- Kettenlaufzeit: ich sehe bei Kunden und bei mir selbst 5000-7000km mit gewachster 
  Kette vs. 2000-2500km mit Öl. ZeroFriction Cycling hat das gut dokumentiert.

Macht ihr auch Heißwachs? Was sind eure Erfahrungen?
```

## 7. Google Search Console (nachdem Domain aktiv ist)

1. Geh auf search.google.com/search-console
2. "Property hinzufügen" → URL-Präfix → `https://waxcelerate.de`
3. Verifizierungsmethode: **HTML-Meta-Tag**
4. Du bekommst einen Code wie: `<meta name="google-site-verification" content="abc123XYZ">`
5. **Öffne** `/Users/lucateichmann/Claude Playground/waxcelerate-site/index.html`
6. **Suche** nach: `<!-- <meta name="google-site-verification" content="XXXXX" /> -->`
7. **Ersetze** durch: `<meta name="google-site-verification" content="DEIN_ECHTER_CODE" />`
   (die Kommentarzeichen `<!--` und `-->` entfernen, XXXXX durch deinen Code ersetzen)
8. Datei speichern → `git add index.html && git commit -m "Add Google verification" && git push`
9. In Search Console auf "Bestätigen" klicken
